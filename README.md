## How to use
1. Open the app  
2. Press **Insert** to toggle the overlay  
3. Enjoy!

---

## About
- Created by **nolife.code**  
- Inspired by **Crosshair X** and **Crosshair V2**  
- Built for fun and learning, lightweight and minimal

---

## Features
- Minimal overlay crosshair  
- Toggle visibility with **F1**  
- MacOS-style / flat design  
- Works on Windows
